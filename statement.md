# Project Statement: [Quiz]

## 1. Problem Statement
Trying to create a  simple, interactive command-line quiz game built with Python. The player answers a series of multiple-choice questions to earn points. The game features a "sudden death" mechanic where one wrong answer ends the game, and it tracks the final score.

## 2. Scope of the Project

### Included (In Scope)
* Implementing core quiz logic.
* Displaying questions and options clearly in the terminal.
* Allowing integer input for choices.

### Excluded (Out of Scope)
* Connecting to an external database for questions.
* Building a Graphical User Interface (GUI) or a web version.

## 3. Target Users
* **Casual Programmers:** Developers or students who want a quick distraction or study break accessible directly from their terminal window.
* **Educators/Trainers:** Developers or students who want a quick distraction or study break accessible directly from their terminal window.

## 4. High-Level Features
1.  **Quiz Presentation:** Display multiple-choice questions dynamically from an internal data structure.
2.  **Scoring System:** Increment the user's score by 10 points for each correct answer.
3.  **Game Termination:** Immediately end the game upon an incorrect answer.
